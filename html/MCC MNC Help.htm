Mobile Country Code (MCC) and Mobile Network Code (MNC) Help

 

To send an Operator Logo using Nokia Smart Messaging, you must supply the GSM Mobile Country Code (MCC) and Mobile Network Code (MNC) of the mobile phone operator to which the mobile phone is subscribed.  The logo will only be displayed when the phone is registered to that operator, and not when roaming on the service of another operator.

 

The GSM Mobile Country Code (MCC) is different from the international dialing country code.  It is a 3 digit code that uniquely identifies a country to the global GSM network.

 

The GSM Mobile Network Code (MNC) is a 2 digit code that uniquely identifies a GSM operator within a particular country.

 

The link below to the web site of the GSM Association contains information about the MCC and MNC values assigned to different countries and mobile operators.  Follow the link to the GSM Association web site, select your country from the list of countries, and then select the link that provides Network Information about your operator.  The Network Code value displayed for your operator will consist of the MCC followed by the MNC.

 

The GSM Association web site is at http://www.gsmworld.com/roaming/gsminfo/index.shtml.
